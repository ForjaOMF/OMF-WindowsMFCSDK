CVideoCallReceiver::SetEndCallCB

Sets the function that will be called when a videocall has ended

void SetEndCallCB(tagCBEndCall pEndCall);

Parameters
pEndCall
  Specifies the address of the application-supplied callback function

Return
None

Remarks
pEndCall function need not be named EndCall, but it must be declared as follows:

static void EndCall(DWORD dwCookie, CVideoCall* pVCall);
