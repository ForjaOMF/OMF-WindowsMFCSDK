CSMSReceiver::CSMSReceiver

Connects to POP server

CSMSReceiver(tagCBMessages pCallbackFunction, DWORD dwCookie, CString csServer, CString csLogin, CString csPassword);

Parameters
pCallbackFunction
  Specifies the address of the application-supplied callback function that will process the messages

dwCookie
  Address of the callback function owner

csServer
  String with POP server

csLogin
  String with POP user's login

csPassword
  String with POP user's password

Return
None

Remarks
pCallbackFunction function need not be named ReceptionFinished, but it must be declared as follows:

void CALLBACK EXPORT ReceptionFinished(DWORD dwCookie, CStringList& slMessages);

where slMessages contains the list of received SMS messages.
