CSMS20::Connect

Connects to SMS2.0 service

CMapStringToOb* Connect(CString csLog, CString csNickname);

Parameters
csLog
  String with user's telephone number

csNickname
  String with the nickname that we want to use (needed only first time)

Return
Contact list of the user indexed by UserId.

Each element in the list is a CSMS20Contact*.

Below you can see a sample code to traverse the list: 
  CSMS20Contact* pContact;
  CString csKey;
  for(POSITION pos=pmsoAddressBook->GetStartPosition();pos;)
  {
  	pmsoAddressBook->GetNextAssoc(pos,csKey,(CObject*&)pContact);
  	if(pContact)
  	{
  		// Do whatever you need with pContact
  	}
  }
