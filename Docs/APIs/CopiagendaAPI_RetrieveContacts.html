CCopiagenda::RetrieveContacts

Performs contact list retrieval

CMapStringToOb* RetrieveContacts(CString csLogin, CString csPwd);

Parameters
csLogin
  String with user's telephone number

csPwd
  String with user's password

Return
Contact list stored for the user. The list is indexed by contact name. Each element in the list is a CStringArray*. Name and telephone number of the contact will be stored in position 3 and 11 of the array respectively.

Below you can see a sample code to traverse the list: 
  CStringArray* psaContact;
  CString csKey;
  for(POSITION pos=pmsoAddressBook->GetStartPosition();pos;)
  {
  	pmsoAddressBook->GetNextAssoc(pos,csKey,(CObject*&)psaContact);
  	if(psaContact)
  	{
  		// psaContact[3] is contact's name
  		// psaContact[11] is contact's telephone number
  	}
  }
