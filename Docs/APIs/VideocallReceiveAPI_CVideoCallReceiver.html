CVideoCallReceiver::CVideoCallReceiver

Creates a new VideoCallReceiver object

CVideoCallReceiver(CString csLogin, CString csPassword, CString csLocalIP, CString csPath, DWORD dwCookie);

Parameters
csLogin
  String with POP user's login

csPassword
  String with POP user's password

csLocalIP
  String with your local IP address. If you leave this parameter empty, default connection address will be used

csPath
  Path where received audio and video files will be stored

dwCookie
  Address of the callback function owner

Return
None
