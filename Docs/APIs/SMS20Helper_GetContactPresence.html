CSMS20Helper::GetContactPresence

Obtains contacts presence status from received XML

CMapStringToString* GetContactPresence(CString csData);

Parameters
csData
  XML text received

Return
List with contacts and their presence status indexed by UserId.

Each element in the list is a CSMS20Contact*.

Below you can see a sample code to traverse the list: 
  CSMS20Contact* pContact;
  CString csKey;
  for(POSITION pos=pmsoAddressBook->GetStartPosition();pos;)
  {
  	pmsoAddressBook->GetNextAssoc(pos,csKey,(CObject*&)pContact);
  	if(pContact)
  	{
  		// Do whatever you need with pContact
  	}
  }
