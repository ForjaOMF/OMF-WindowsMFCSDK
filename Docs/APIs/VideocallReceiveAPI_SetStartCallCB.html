CVideoCallReceiver::SetStartCallCB

Sets the function that will be called when you receive a new videocall

void SetStartCallCB(tagCBStartCall pStartCall);

Parameters
pStartCall
  Specifies the address of the application-supplied callback function

Return
None

Remarks
pStartCall function need not be named StartCall, but it must be declared as follows:

static void StartCall(DWORD dwCookie, CVideoCall* pVCall);
